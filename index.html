Projects

AutoCat: A Hashcat Frontend

AutoCat is a command-line interface (CLI) tool designed to streamline the use of Hashcat, a powerful password recovery and cracking utility. It provides an interactive and user-friendly frontend that simplifies the process of configuring and launching Hashcat attacks. By guiding the user through the selection of hash types, attack modes, and compute devices, AutoCat eliminates the need to memorize complex command-line flags. This project demonstrates a practical application of shell scripting to create a more accessible and efficient workflow for a widely used cybersecurity tool.

Carbon Footprint Calculator

The Carbon Footprint Calculator is a full-stack web application designed to raise awareness about individual environmental impact. It allows users to calculate their carbon footprint by inputting data related to their daily activities, including transportation, food consumption, and energy usage. The application is built with a React frontend that provides a user-friendly interface for data entry and visualization. The backend is powered by Flask and Python, which processes the data, calculates the carbon footprint, and stores it in a MongoDB database for historical tracking. The results are presented in a clear and intuitive manner, with a pie chart that breaks down the carbon footprint by category and provides actionable tips for reducing it. This project demonstrates proficiency in full-stack development, including frontend and backend technologies, database management, and data visualization.

Real-Time Sign Language Recognition

This project is a real-time sign language recognition system that leverages computer vision and deep learning to translate sign language gestures into text. The system uses the MediaPipe library to perform real-time hand and body landmark detection from a webcam feed. The extracted landmark data is then fed into a Long Short-Term Memory (LSTM) neural network, which is trained to recognize and classify different sign language gestures. This project demonstrates a practical application of machine learning for accessibility, with the potential to bridge the communication gap between the deaf and hearing communities. The use of an LSTM network is particularly well-suited for this task, as it can effectively model the temporal sequences of sign language gestures.

Gamified Samvidhan: An Interactive Learning Platform

Gamified Samvidhan is a full-stack web application that provides a unique and engaging platform for learning about the Indian Constitution. The project transforms the process of studying a complex legal document into an interactive and enjoyable experience. It features a variety of gamified elements, including daily quizzes, trivia games, and a multiplayer mode, to motivate users and enhance their understanding of the constitution. The platform also includes user authentication, progress tracking, and a rewards system to encourage continued learning. The frontend is built with React and Tailwind CSS, while the backend is powered by Node.js, Express, and MongoDB. This project showcases the ability to design and develop a comprehensive e-learning platform with a focus on user engagement and interactive learning.
